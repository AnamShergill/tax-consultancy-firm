# Background Pattern Image

Please download a subtle pattern or texture image and save it as:

**public/images/bg-pattern.jpg**

## Recommended Image Types:

### Subtle Patterns (Best Choice)
- Light geometric patterns
- Subtle textures (paper, fabric, concrete)
- Minimal noise patterns
- Light grid patterns

### Free Pattern Resources:

## Image Specifications:
- Format: JPG or PNG
- Recommended size: 1920x1080px or smaller (patterns can tile)
- File size: Under 200KB (should be very light)
- Style: Subtle, light, minimal (won't distract from content)

## Good Pattern Examples:
- Light paper texture
- Subtle diagonal lines
- Minimal dots pattern
- Light geometric shapes
- Fabric texture
- Concrete texture (very light)

## Important Notes:
- The pattern should be VERY subtle
- Light colors work best (white, light gray, beige)
- The CSS applies a 95% white overlay, so the pattern will be barely visible
- This creates depth without being distracting

If you don't want a pattern, you can leave this as is and the website will use a solid color background.
